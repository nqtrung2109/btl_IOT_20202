* Cách cài đặt và sử dụng hệ thống Chat
- Cách 1: 
B1: Cài đặt phần mềm Visual Studio Code (hoặc các IDE khác như Atom,WebStorm...), sau đó cài đặt NodeJs.
B2: Sau khi đã giải nén file EE4266-129144-Nhom-12, mở phần mềm Visual Studio Code lên
vào File và mở folder EE4266-129144-Nhom-12.
B3: Các thư viện được dùng trong hệ thống Chat đã được tải và có sẵn trong folder node_modules
lên không cần tải và cài đặt lại. Mở New Terminal lên và gõ "node server.js"
B4: Lên trên Web vào truy cập vào đường dẫn "http://localhost:3000" để mở hệ thống chat lên.
